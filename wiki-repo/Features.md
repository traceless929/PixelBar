# 功能支持范围

## 已支持

| 功能 | CLI / SDK | 桌面客户端 |
|------|-----------|------------|
| 文字 + 布局（静态/滚动） | ✅ | ✅ |
| RGB 灯光（6 种模式） | ✅ | ✅ |
| 时钟样式 1–11 | ✅ | ✅ |
| 频谱样式 1–4 | ✅ | ✅ |
| 像素屏主题色 | ✅ | ✅ |
| **QQ 音乐动态歌词** | — | ✅（[说明](Lyrics-QQMusic)） |
| 开机启动 / 托盘 | — | ✅ |
| 使用引导 | — | ✅ |

### 动态歌词（桌面客户端）

- 本地解密 QQ 音乐 `*_qm.qrc` 缓存
- SMTC 播放进度同步、时间偏移微调
- 长句滚动与方向可选
- 解密结果落盘 `%LocalAppData%\PixelBar\LyricCache`
- QQ 音乐桌面歌词回退

## 不支持

以下须使用官方 **EDIFIER TempoHub / Connect**：

- 游戏、办公、阅读、宠物、表情包、赛博等个性场景模板
- 自创空间图片/动画（`0x17` 帧上传）
- 热点新闻、天气预报
- TempoHub 新版 EF 时钟等需帧上传的样式

## 设备要求

- **设备**：漫步者花再 Halo PixelBar
- **VID/PID**：`0x2D99` / `0xA106`
- **系统**：Windows 10+ x64
- **连接**：USB

协议细节见仓库 [README · 协议格式](https://github.com/traceless929/PixelBar#协议格式)。
