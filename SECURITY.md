# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| latest release / `main` | ✅ |

## Reporting a Vulnerability

请勿在公开 Issue 中披露可利用的安全问题。请通过 GitHub **Private vulnerability reporting**（仓库 Security 标签页）或维护者邮箱私下联系。

本项目为社区 HID 互操作工具，**不包含**官方固件更新通道；请勿将本仓库用于未授权访问他人设备或绕过商业软件授权。
