# 路线图

以下为计划中的功能；优先级与实现方式可能随逆向进展调整。

欢迎 [Issue / PR](https://github.com/traceless929/PixelBar/issues)；也鼓励开发者**直接基于 [PixelBar.Sdk](SDK-Development)** 先行实现原型，成熟后再合入主线。

## 计划功能

| 方向 | 目标 | 说明 |
|------|------|------|
| **RGB 神光同步** | 氛围灯与 PC 灯效生态联动 | 对接华硕 Aura Sync、微星 Mystic Light、技嘉 RGB Fusion 等，将系统/游戏/屏幕取色同步到 PixelBar 氛围灯 |
| **动态歌词 · 扩展** | 更多播放器 | **v0.0.2 已支持 QQ 音乐**（见 [Lyrics-QQMusic](Lyrics-QQMusic)）；后续：网易云音乐等 |
| **协议补全** | 游戏/宠物/0x17 模板 | 依赖对 TempoHub `0x17` 帧上传流程的进一步逆向 |
| **体验优化** | 托盘、引导、歌词微调等 | 持续迭代 |

## 预期链路（草案）

### RGB 神光同步

```
系统 RGB 源（Aura SDK / 屏幕取色 / 游戏钩子）→ 颜色与模式映射 → PixelBar RGB 灯光包（0x6B）
```

### 动态歌词（QQ 音乐 · 已实现）

```
QQ 音乐 qrc 缓存 → 本地解密 → SMTC 进度 → PixelBar 文字协议 → 像素屏
```

详见 **[QQ 音乐动态歌词](Lyrics-QQMusic)**。
